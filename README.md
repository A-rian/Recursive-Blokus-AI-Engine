# Recursive Blokus AI Engine

A Python AI engine for a simplified version of Blokus using recursive game-tree search, heuristic evaluation, simulations, and Plotly visualizations.

## Features

- Recursive game-tree generation
- Heuristic-based AI decision-making
- Multiple AI difficulty levels
- Automated game simulations
- Win-rate and score analysis
- Plotly-based visualizations

## Technologies

Python, Plotly, Object-Oriented Programming, Recursion, Game Trees, Heuristic Search
